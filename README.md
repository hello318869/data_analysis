# 交互式数据分析系统

基于 Flask 的数据清洗与分析 Web 应用，支持缺失值处理、异常值检测、规则集清洗等功能。

## 功能特点

- 📊 **数据清洗**：缺失值填充（均值/中位数/众数/自定义）、异常值检测（IQR/Z-Score）
- 🧹 **一键自动清洗**：数值列中位数填充 + 分类列众数填充 + IQR 截断
- 📋 **规则集管理**：保存/加载清洗规则，预设模板快速应用
- 📁 **文件上传**：支持 CSV / Excel 格式
